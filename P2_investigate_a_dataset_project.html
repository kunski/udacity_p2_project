def correlation(x, y):
    std_x = (x - x.mean()) / x.std(ddof = 0)
    std_y = (y - y.mean()) / y.std(ddof = 0)
    return (std_x * std_y).mean()

Limitations¶

I would like to mention some limitations for this analysis in followings;

Limitation of Dataset

According to the Wikipedia page of Titanic[1], there are 2,224 passengers on board when that accident happened, but this data set only contains data of 891 passengers.
It can be possbible that there might be a sample bias for that data set, for example, data might be mainly collected by the passengers who survived or more from female passengers, and passenger who did survived but not on the data set because of some private reasons. (Or there could be some passenger who were not even the list.)
Therefore it might be difficult to conclud that these factors really made every signle passengers survived or not from this data set.
